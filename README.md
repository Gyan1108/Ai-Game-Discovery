# Ai-Game-Discovery
Game Discovery AI is a full-stack, production-ready intelligent recommendation platform designed to help users discover video games using natural language search instead of rigid filters. Unlike traditional game discovery platforms that rely on predefined dropdowns and exact keyword matching, this system leverages modern NLP, vector similarity search, and agentic reasoning to interpret user intent and retrieve semantically relevant games from a large-scale dataset of over 470,000+ titles.

The project integrates Natural Language Processing, Machine Learning, Vector Databases, Cloud-native deployment, and modern frontend engineering to build a scalable, real-world AI application that mimics how humans naturally describe preferences

## Core Motivation & Problem Statement

### Current game discovery systems suffer from:

- Over-reliance on static filters (genre, price, platform)

- Poor handling of vague or descriptive queries

- Inability to combine semantic meaning + metadata reasoning

- Limited scalability when datasets grow large

### This project addresses these issues by:

- Translating free-form user input into semantic embeddings

- Querying a large-scale vector database for meaning-based similarity

- Applying agentic reasoning layers for intent analysis, ranking, and explanation

- Delivering results with contextual explanations and visual enrichment (posters)
